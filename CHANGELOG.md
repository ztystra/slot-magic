# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-21

### Added
- Telegram bot with keyboard menu
- Service selection with prices
- Date and time slot selection
- Booking creation with name/phone
- Automatic reminders (24h and 2h)
- Booking cancellation
- Configurable services and work hours
- JSON-based data storage
- GitHub Actions CI/CD
- Issue templates (bug + feature)
- Contributing guide
- Security policy
- Code of conduct
- Star History badge
